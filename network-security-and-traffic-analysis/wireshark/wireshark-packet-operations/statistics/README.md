## Protocol Hierarchy

&#x20;**"Statistics --> Protocol Hierarchy"**&#x20;



## Conversations

"Statistic --> Conversations"

Conversation represents traffic between two specific endpoints. There are fiive base formats; ethernet, IPv4, IPv6, TCP and UDP.



## Endpoints

The endpoints option is similar to the conversations option. The only difference is that this option provides unique information for a single information field (Ethernet, IPv4, IPv6, TCP and UDP ).

"Statistics --> Endpoints"
